### Task 2: Create Export Utility (`src/lib/export.ts`)

Core functions:
- `exportCSV(data, columns, filename)` â€” BOM-prefixed CSV, triggers download
- `exportXLSX(data, columns, filename)` â€” uses xlsx lib, auto-sized columns
- `exportPDF(data, columns, filename, pdfConfig?)` â€” AtlasPM branded PDF with header, stats, table, page numbers
- `exportChartPNG(element, filename)` â€” html2canvas capture + download

**Column config type:**
```typescript
export interface ExportColumn {
  key: string;
  label: string;
}
export interface PDFConfig {
  title: string;
  subtitle?: string;
  stats?: { label: string; value: string }[];
}
```

**CSV:** Map data through columns, join with commas, escape quotes, BOM prefix, Blob download.

**XLSX:** Reuse existing pattern from `excel-export.ts` â€” `json_to_sheet`, auto column widths, single sheet.

**PDF layout:**
- Page: A4 landscape for tables
- Header: "ATLAS PM" (bold, 18pt) | report title (14pt) | date (10pt, right-aligned)
- Stats row: if pdfConfig.stats provided, render as label:value pairs in a gray row
- Data table: autoTable plugin, alternating row fills (#f8f9fa / white), header bg #0a1628, text white
- Footer: "Page X of Y" centered, "Generated by AtlasPM" right-aligned

**Chart PNG:** `html2canvas(element)` â†’ `canvas.toBlob()` â†’ download.

### Task 3: Create ExportButton Component (`src/components/ui/export-button.tsx`)

A dropdown button using the existing `outline` variant from `button.tsx`. Shows a popover with format options on click.

**Props:**
```typescript
interface ExportButtonProps {
  data: Record<string, any>[];
  filename: string;
  formats?: ("csv" | "xlsx" | "pdf")[];
  columns?: ExportColumn[];
  pdfConfig?: PDFConfig;
  chartRef?: React.RefObject<HTMLElement>;
  disabled?: boolean;
}
```

**Behavior:**
- Click toggles a dropdown below the button
- Each format option calls the corresponding export function
- If `chartRef` provided, shows additional "Chart as PNG" option
- Auto-derives columns from `Object.keys(data[0])` if `columns` not provided
- Shows `Download` icon from lucide-react
- Close dropdown on click outside (useEffect listener)
- Disabled state when `data` is empty
